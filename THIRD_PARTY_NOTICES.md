LockCat includes third party assets and tools whose licenses are kept with the source tree.

## Fusion Pixel Font

LockCat uses Fusion Pixel Font for its pixel-style UI text.

The font license files are included under:

- `src/LockCat/Assets/Fonts/FusionPixelFont-OFL.txt`
- `src/LockCat/Assets/Fonts/FusionPixelFont-LICENSE-OFL.txt`
- `src/LockCat/Assets/Fonts/FusionPixelFont-Upstream-LICENSE/`

Please keep these notices when redistributing LockCat or modified builds.

## Pixel Artwork

LockCat's app icons, setup illustrations, UI pixel elements, and desktop cat sprites are included in this repository for use with LockCat.

If you reuse the artwork outside LockCat, please check the repository license and keep attribution to this project.
